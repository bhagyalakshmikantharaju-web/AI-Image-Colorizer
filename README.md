AI Image Colorizer 🎨

📌 Project Description

AI Image Colorizer is a Python-based project that converts black and white images into colorized images automatically.
The system uses image processing techniques with OpenCV to apply color mapping to grayscale images.

This project demonstrates how Artificial Intelligence and computer vision techniques can enhance old black and white photos by adding colors.

---

🚀 Features

- Converts grayscale images into color images
- Uses OpenCV image processing
- Simple and easy to run
- Works with JPG and PNG images
- Suitable for beginners learning AI and Python

---

🛠 Technologies Used

- Python
- OpenCV
- NumPy
- Matplotlib

▶ How to Run the Project

Step 1: Install Required Libraries

pip install opencv-python
pip install numpy
pip install matplotlib

Step 2: Run the Program

python colorizer.py

Step 3: Upload or place a grayscale image in the project folder.

The program will display:

- Original Image
- Colorized Image

📸 Output

The system converts grayscale images into visually enhanced color images.
